# 下载 WinIsland

WinIsland 目前正在积极开发中。你可以在下方下载最新的每日预览版 (Nightly Build)。

## 预览版 (Nightly)

预览版会在我们每次更新代码时自动生成。它们包含最新的功能，但可能不如官方正式版稳定。

[下载最新预览版 (.exe)](https://github.com/Neamyoo-dev/WinIsland/releases/download/nightly/WinIsland.exe)

---

### 安装说明

1. 下载 `WinIsland.exe`。
2. 将其移动到你想要保存的文件夹中。
3. 运行该可执行文件。
4. （可选）你可以添加一个快捷方式到你的 `启动` (Startup) 文件夹中，让它开机自启。

### 系统要求

- Windows 10 或 11
- 64 位架构
- 支持 Skia 的现代显卡（大多数都支持）
